# 快速开始指南

本指南帮助您在 30 分钟内完成点餐小程序的部署。

## 前提条件

- ✅ 已有微信小程序账号（个体工商户或企业类型）
- ✅ 已完成小程序认证
- ✅ 已安装微信开发者工具

## 部署步骤

### 步骤 1：导入项目

1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择项目目录
4. 填入你的 AppID
5. 点击「导入」

### 步骤 2：配置云开发

1. 点击工具栏「云开发」按钮
2. 创建云环境（选择「按量付费」）
3. 复制环境 ID
4. 修改 `cloudbaserc.json` 中的 `envId`

### 步骤 3：部署云函数

1. 右键点击 `cloudfunctions` 文件夹
2. 选择「同步云函数列表」
3. 等待所有云函数部署完成（约 2-3 分钟）

### 步骤 4：初始化数据库

在调试器控制台执行：

```javascript
wx.cloud.callFunction({
  name: 'initDatabase'
}).then(res => console.log('初始化结果：', res))
```

### 步骤 5：配置商家权限

1. 登录小程序
2. 在云开发控制台打开 `users` 集合
3. 复制你的 `_openid`
4. 在 `merchantWhitelist` 集合添加记录：

```json
{
  "openid": "你的openid",
  "status": 1
}
```

### 步骤 6：测试运行

1. 点击「编译」按钮
2. 测试基本功能
3. 点击「预览」进行真机测试

## 核心功能说明

### 用户端

| 功能 | 说明 |
|-----|------|
| 扫码点餐 | 扫描桌号二维码自动获取桌号 |
| 菜品浏览 | 分类浏览、热门推荐 |
| 购物车 | 添加/修改数量 |
| 提交订单 | 堂食/自取/外卖三种模式 |
| 订单管理 | 列表、详情、支付、取消 |

### 商家端

| 功能 | 说明 |
|-----|------|
| 订单管理 | 接单、出餐、完成、取消 |
| 菜品管理 | 增删改查、上下架 |
| 桌号管理 | 生成二维码 |
| 自动接单 | 开关配置 |
| 蓝牙打印 | 小票、后厨单 |

## 常用配置

### 修改店铺信息

在云开发控制台修改 `shopInfo` 集合：

```json
{
  "name": "你的店铺名称",
  "address": "店铺地址",
  "phone": "联系电话",
  "businessHours": "10:00-22:00"
}
```

### 开启自动接单

在 `shopInfo` 集合中设置：

```json
{
  "autoAcceptOrder": true
}
```

### 配置支付功能

1. 申请微信支付商户号
2. 关联小程序和商户号
3. 修改 `cloudfunctions/createPayment/index.js` 中的 `MCH_ID`
4. 在云开发控制台配置支付参数

详见 [完整部署指南](./DEPLOYMENT.md#第五步支付配置)

## 注意事项

1. **必须使用个体工商户或企业账号**，个人账号无法开通支付
2. **云开发免费额度有限**，注意监控使用情况
3. **定期备份数据**，防止数据丢失
4. **测试充分后再上线**，避免影响用户体验
