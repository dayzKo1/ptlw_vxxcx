# 快速开始指南

本指南帮助您在 5 分钟内完成点餐小程序的部署。

## 前提条件

- ✅ 已有微信小程序账号
- ✅ 已安装微信开发者工具

## 快速部署

### 1. 导入项目

1. 打开微信开发者工具
2. 导入项目，填入 AppID
3. 点击「云开发」创建环境

### 2. 配置环境

修改 `app.js` 中的云开发环境 ID：

```javascript
wx.cloud.init({
  env: '你的云开发环境ID',
})
```

### 3. 部署云函数

1. 右键 `cloudfunctions` 文件夹
2. 选择「同步云函数列表」
3. 等待部署完成（约 1-2 分钟）

> 本项目使用模块化云函数，仅 8 个云函数

### 4. 初始化

在控制台执行：

```javascript
wx.cloud.callFunction({ 
  name: 'initDatabase', 
  data: { force: true, testOrders: true } 
}).then(r => console.log(r))
```

这将自动：
- 创建数据库集合
- 初始化示例数据
- 将你添加为商户

### 5. 测试

刷新页面，开始测试！

## 常见问题

**Q: 登录失败？**
- 检查云开发环境 ID 是否正确
- 确认云函数已部署

**Q: 商户后台无法访问？**
- 重新执行 `initDatabase` 初始化

**Q: 打印机连接失败？**
- 确认打印机电源和蓝牙已开启
