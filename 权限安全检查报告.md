## 安全最佳实践

### 1. 云函数权限验证模式

```javascript
// 商户权限验证
async function checkMerchantPermission() {
  const wxContext = cloud.getWXContext()
  const res = await db.collection('merchantWhitelist')
    .where({ openid: wxContext.OPENID, status: 1 })
    .get()
  return res.data.length > 0
}

// 用户数据所有权验证
const data = await db.collection('xxx').doc(id).get()
if (data.data._openid !== wxContext.OPENID) {
  return { success: false, message: '无权限' }
}
```
