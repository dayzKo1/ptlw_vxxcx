# 外卖配送功能说明

## ✅ 已完成的功能

### 1. 配送方式

支持三种配送方式：
- **堂食**：店内用餐，需要桌号
- **自取**：到店自取
- **配送**：外卖配送

### 2. 购物车页面优化

#### 新增功能
- ✅ 配送方式切换
- ✅ 配送地址选择
- ✅ 配送费计算
- ✅ 商品金额和配送费分别显示
- ✅ 配送时必须选择地址的验证

#### 费用计算
- **堂食/自取**：商品金额
- **配送**：商品金额 + 配送费

### 3. 地址管理

#### 页面路径
- 地址列表：`packageUser/addressList/addressList`
- 地址编辑：`packageUser/addressEdit/addressEdit`

#### 功能
- ✅ 添加新地址
- ✅ 编辑已有地址
- ✅ 删除地址
- ✅ 设置默认地址
- ✅ 表单验证

#### 表单字段
- 联系人姓名
- 手机号（11位数字验证）
- 所在地区（省市区选择）
- 详细地址
- 设为默认地址

### 4. 订单数据结构

```javascript
{
  orderNo: '订单号',
  orderType: 'T/P/D',          // T=堂食, P=自取, D=外卖
  tableNumber: '桌号',         // 堂食时有值
  items: [...],
  totalPrice: 56.00,
  remark: '备注信息',
  deliveryMode: 'pickup/delivery',
  addressId: '配送地址ID',     // 外卖时有值
  status: 0
}
```

### 5. 订单号规则

- 堂食订单：`T桌号-序号`，如 `T05-001`
- 自取订单：`P序号`，如 `P001`
- 外卖订单：`D序号`，如 `D001`

## 🔧 云函数

### 地址相关
- `getAddresses` - 获取地址列表
- `addAddress` - 添加新地址
- `updateAddress` - 更新地址
- `deleteAddress` - 删除地址
- `setDefaultAddress` - 设置默认地址

### 订单相关
- `createOrder` - 创建订单（支持配送模式）
- `getUserOrders` - 获取用户订单

## 🎯 使用流程

### 堂食点餐
1. 扫描桌号二维码
2. 选择菜品加入购物车
3. 确认订单信息
4. 提交订单

### 自取/外卖
1. 进入点餐页面
2. 选择菜品加入购物车
3. 进入购物车，选择配送方式
4. 选择配送地址（外卖时）
5. 提交订单
